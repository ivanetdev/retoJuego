# 📚 Juego de Parejas

Aplicación web de un juego de memoria desarrollada con **JavaScript**, donde el usuario debe encontrar las parejas de cartas en el menor tiempo posible.

## 🌐 Demo online

Puedes probar el juego aquí:

👉 https://ivanetdev.github.io/retoJuego/



## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- GitHub Pages



## 🚀 Instalación y uso

1️⃣ Clonar el repositorio:

bash
git clone https://github.com/ivanetdev/retoJuego.git

2️⃣ Entrar en la carpeta:

bash
cd retoJuego

3️⃣ Ejecutar el proyecto:

Abre el archivo index.html en tu navegador
o accede directamente desde GitHub Pages.
